# Email Service Proposal for Property Price Website

---

**Prepared for:** Client Review  
**Date:** December 11, 2025  
**Project:** Property Price Static Website  

---

## Executive Summary

For the Property Price website, we recommend using **Brevo** (formerly Sendinblue) as the email service provider. Since this is a frontend-only project without a traditional backend server, we have evaluated the best approaches to handle email functionality securely and efficiently.

## Why Brevo?

1. **Cost-Effective** - Generous free tier suitable for project launch
2. **No Backend Required** - Embedded forms work without server code
3. **Enterprise-Grade** - Used by 500,000+ businesses globally
4. **GDPR Compliant** - Important for UK/EU property data
5. **Easy Migration** - Can scale to paid plans as business grows
6. **Reliable Delivery** - High email deliverability rates
